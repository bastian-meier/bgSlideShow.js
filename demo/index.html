BgSlideShow.js

simple background slideshow without dependencies and less than 3kb filesize

A very simple and small script for creating Background-slideshows.
It uses css-classes you define yourself, so it is very flexible.
You can use:

  • Fullsize background-images
  • tiled background-images
  • background-color
  • css3 background-gradients
  • ...

include the js file:

            <script src="BgSlideShow.min.js"></script>

define your background-classes, for example:

            1    .bg-image{
            2        background: url(img/bg-image.jpg);
            3        background-size: cover;
            4    }
            5    
            6    .bg-css3-gradient{
            7        background: -moz-linear-gradient(top, #9DBAF8, #757575);
            8        background: -webkit-gradient(linear, left top, left bottom, from(#757575), to(#9DBAF8));
            9    }
            10   
            11   .bg-image-repeat{
            12       background: url(img/bg-image-repeat.jpg);
            13       background-repeat: repeat;
            14   }
            15   
            16   .bg-color{
            17       background-color: #777;
            18   }

tell BgSlideShow.js what to do:

the first argument is an array of objects with this key - value pairs:

  • className: the name of the background-class you defined
  • duration: how long the class should be active in seconds

the second argument is the config:

  • animationDuration: the duration of the fadeIn animation in sedonds 
            
                122              var bg = new BgSlideShow([
                123                  {className: 'bg-image', duration: 10},
                124                  {className: 'bg-color', duration: 5},
                125                  {className: 'bg-css3-gradient', duration: 10}
                126              ], {animationDuration: 3});

you can also bind single objects:

            bg.addAnimation({className: 'bg-image-repeat', duration: 10});

start stop methods

the animation starts on class initialisation
but there are startLoop() and stopLoop() methods available. 

            bg.start();
            
            bg.stop();
