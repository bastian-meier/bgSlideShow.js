BgSlideShow.js

simple background slideshow without dependencies and less than 3kb filesize

A very simple script for creating Background-slideshows.
It is base on css-classes you define yourself, so it is very flexible.
You can use:

  • Fullsize background-images
  • tiled background-images
  • background-color
  • css3 background-gradients

include the js and the css files:

            1       <link href="BgSlideShow.min.css" rel="stylesheet" />
            2       <script src="BgSlideShow.min.js"></script>

define your background-classes, for example:

            1    .bg-image{
            2        background: url(img/bg-image.jpg);
            3        background-size: cover;
            4    }
            5    
            6    .bg-css3-gradient{
            7        background: -moz-linear-gradient(top, #9DBAF8, #757575);
            8        background: -webkit-gradient(linear, left top, left bottom, from(#757575), to(#9DBAF8));
            9    }
            10   
            11   .bg-image-repeat{
            12       background: url(img/bg-image-repeat.jpg);
            13       background-repeat: repeat;
            14   }
            15   
            16   .bg-color{
            17       background-color: #777;
            18   }

tell BgSlideShow.js what to do:

className: the name of the background-class you defined
duration: how long the class should be active (in ms) 

            
            72       <script>
            73           window.onload = function ()
            74           {
            75               var bg = new BgSlideShow([
            76                   {className: 'bg-image', duration: 4000},
            77                   {className: 'bg-color', duration: 4000},
            78                   {className: 'bg-css3-gradient', duration: 4000}
            79               ]);
            81           }
            82       </script>

you can also bind a single object:

            
            80               bg.addAnimation({className: 'bg-image-repeat', duration: 4000});
